# Dibuy 
[https://dibuy.netlify.app/](https://dibuy.netlify.app/)
### Digital stock integrator

![Home page](https://res.cloudinary.com/dzivm8ve4/image/upload/v1678960092/Screenshot_2023-03-16_145831_vhrqiq.png?raw=true)


#### ● Dibuy is a digital stock exchanging MERN stack project which is intended to benifit the small scale farmers and vendors by providing internal stock integration,products selling feasibilities with zero intermediaries.

#### ● We have implemented different routes with different features to enable good user experience. Our site contains Home route,Seller corner,Dashboard,Profile section,Products route,History route,Cart route and Feedback sections.
#### ● A lot of REST API's are written in the backend to perform different operations in the database to make it realtime.
#### ● A user can register and login with OTP authentication. An authorized user can buy products and also can sell products by registering as a seller.
![Login](https://res.cloudinary.com/dzivm8ve4/image/upload/v1678960091/Screenshot_2023-03-16_145729_vturrc.png?raw=true)
#### ● A seller can add products,track products and check thier sales history with realtime dashboard.
![DashBoard](https://res.cloudinary.com/dzivm8ve4/image/upload/v1678960092/Screenshot_2023-03-16_145949_vdzpkv.png?raw=true)
#### ● They can also manage their account and contributed products details.
#### ● Any user can buy products at the lowest price and will be notified with corresponding emails.
![Products](https://res.cloudinary.com/dzivm8ve4/image/upload/v1678960098/Screenshot_2023-03-16_145906_idgoym.png?raw=true)
#### ● A user will be suggested with top-rated products and sales tips to make them aware.

#### ● Account route provides security with unique QR and avails to make changes of password etc.
![Account](https://res.cloudinary.com/dzivm8ve4/image/upload/v1678960094/Screenshot_2023-03-16_150013_ieam8e.png?raw=true)
